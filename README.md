## Score Keeper App
It is a tool to keep track of progress made by two players while playing a game.

## Thechnologies 
This project is created with
1. HTML
2. CSS
3. Javascript

## Framework
[Bulma](https://bulma.io/)

## To run this project
* Download Project
* Run .html file in browser
